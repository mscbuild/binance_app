# Binance Price List Web-App (Cryptocurrency Price Prediction)

<img width="1351" height="470" alt="23" src="https://github.com/user-attachments/assets/7361294b-efe5-4757-a225-875e4888f54a" />


The app serves as a monitoring tool that alerts users to specific price levels or volatility without the need to manually check charts, similar to the built-in alerts features on the Binance platform.

- **Real-time Tracking:** Connects to Binance.com for up-to-the-second data on trading pairs.
  
- **Custom Alerts:** Users can set specific price targets or percentage changes.
  
- **Desktop Notifications:** Provides immediate alerts when price conditions are met.
  
- **Targeted Monitoring:** Allows monitoring of specific coins or overall market 24h performance.
